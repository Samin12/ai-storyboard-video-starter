# Disapproved / Needs Fix

This folder shows the kind of material that should not become the final anchor set.

In this example, the main reason to reject a frame would be identity drift:

- Samin's face looks too different.
- The cup changes shape or color.
- The kitchen no longer feels like the same room.
- The clock or apron continuity breaks.

Rejected frames should stay here so the next attempt can learn from them.
